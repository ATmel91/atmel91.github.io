Discord secure encryption

What is Oskamp? Oskamp is an encryption tool meant for Discord. It allows you to send encrypted messages to only certain people. Meaning even if somebody else has Oskamp, users have to be let in on the conversation first. People can be invited into conversations, voted out, and/or forced out of the convo by an admin with Oskamp, among many other features.

I have decided to create a custom python cryptography library for Oskamp in order to streamline everything. Currently my library (called eccrypto) is under development but as of right now, I have 80% of the library finished. Once the custom lib is completed, I will then start working on the encryption self-bot.
